# YOLO ROS Wrapper 

![Darknet Logo](http://pjreddie.com/media/files/darknet-black-small.png)

**Author** Ayush Gaud

### Dependencies
1. CUDA  
2. ROS Kinetic

### Instructions

```
git clone https://github.com/ayushgaud/darknet_ros.git
git submodule init
git submodule update

catkin_make --pkg darknet_ros

```
